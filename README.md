This is the final project of 6 sprints of training in Yandex Praktikum.

## Technologies
Project is created with:
* Java 11 
* JUnit 4.13.2
* maven 3.9.2
* Allure 2.24.0
* RestAssured 4.4.0
* gson 2.8.9

## Setup
To run this project:

```
$ cd Sprint_7
$ mvn compile 
$ mvn clean test
$ mvn allure:serve
```
